Guide

So, you want to learn a language to be able to speak to people all across the world. You've come to the right place! We don't have any fancy linguistics jargon here; only simple lessons designed for the layman.

1. Sounds and spelling

Learn how to read and write Baslamo.

2. How to make a sentence

Learn how to make basic sentences.

3. Making new words

Learn how to create new words by using compounds and kesekmolas.

4. Incomplete, complete, and future actions

Learn how to talk about when actions happened.

5. Understanding doskamolas

Learn how to use doskamolas, a collection of very useful words.

6. How to make a sentence, but harder

Learn how to make more complex sentences.

7. Names of people and places

Learn how to fit names into Baslamo's sound system.

8. Introduce yourself

Make an introduction in Baslamo, including your name, where you're from, and your hobbies.
